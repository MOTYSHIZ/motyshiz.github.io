InVRasion (Working Title)

InVRasion (Working Title) is a work in progress virtual reality game for the HTC VIVE. It is an asynchronous multiplayer game in which a single VR player competes against up to four other players who use co-operative splitscreen. The VR player plays as a giant, city destroying monster, while the other players play as soldiers who attempt to defeat the monster before it destroys an objective. InVRasion has been in development since January of 2017 as a Senior Project for my degree in Computer Science at Cal Poly Pomona. After receiving academic credit for InVRasion, development has continued as a hobby since March of 2017. Created using Unity3D Game Engine and C#. Try out InVRasion here!

Invisible Heroes

Invisible Heroes is my first Unity programming project. It is a simple game that requires the player to locate an artifact on a procedurally generated map. There is a readme within the linked file that includes more details.
